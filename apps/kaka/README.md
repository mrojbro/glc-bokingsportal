# Kåkå — CSV till Excel

Transform uploaded CSV files into the Kåkå output format. Edit rows in the browser and download as `.xlsx`. All processing runs locally — no server.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```
